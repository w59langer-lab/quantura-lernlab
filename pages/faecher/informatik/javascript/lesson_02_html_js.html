Lektion 02: HTML & JavaScript – Erste Interaktivität

HTML liefert die Struktur, JavaScript macht sie lebendig – wir verknüpfen beides mit einfachen DOM-Operationen.

Lernziele

  • Das DOM als Schnittstelle zwischen HTML-Struktur und JavaScript verstehen.
  • Elemente mit querySelector gezielt auswählen.
  • Mit addEventListener("click", ...) auf Benutzeraktionen reagieren.
  • Textinhalte über textContent oder innerText ändern.
  • Eigenen Mini-Playground nutzen, um HTML & JS auszuprobieren.

Wiederholung: HTML-Grundgerüst

Eine HTML-Seite besteht aus <html>, <head> und <body>. Im <head> stehen Meta-Infos, Titel und Styles; im <body> stehen die sichtbaren Elemente wie Überschriften, Absätze und Buttons.

Wichtige Tags: h1–h6 für Überschriften, p für Absätze, button für Klicks, div und section als Container.

JavaScript Basics im DOM

DOM? Das Document Object Model ist die Baumstruktur, in der der Browser dein HTML darstellt. JS greift auf Knoten zu, liest Werte aus oder ändert sie.

querySelector wählt ein Element per CSS-Selektor: document.querySelector("#meinButton").

addEventListener("click", ...) registriert eine Funktion, die bei einem Klick ausgeführt wird.

textContent / innerText setzen den sichtbaren Text eines Elements: element.textContent = "Neu";.

Mini-Beispiel

Ein Klick auf den Button ändert den Absatztext:

<p id="status">Noch nichts passiert.</p>
<button id="klickMich">Klick mich</button>
<script>
  const status = document.querySelector("#status");
  const btn = document.querySelector("#klickMich");
  btn.addEventListener("click", () => {
    status.textContent = "Button wurde geklickt!";
  });
</script>

Übungen

  1. Ändere den Text im Beispiel auf einen eigenen Satz und teste.
  2. Füge einen zweiten Button hinzu, der den Text wieder zurücksetzt.
  3. Erweitere das Beispiel, sodass ein Klick den Hintergrund eines div einfärbt.
