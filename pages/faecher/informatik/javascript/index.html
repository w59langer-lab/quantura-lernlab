JavaScript

Grundideen: Variablen speichern Werte, Funktionen bündeln Logik, der DOM verbindet JS mit HTML.

Variablen

Mit let oder const legst du Werte ab, z. B. const name = "Mia";.

Funktionen

Funktionen sammeln Schritte: function begruesse() { alert("Hallo!"); }.

DOM

Über den DOM greifst du auf Elemente zu: document.querySelector("h1") wählt die Überschrift.
