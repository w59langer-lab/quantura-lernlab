Lesson 01 — Hello Web

HTML-Struktur, ein bisschen Styling und ein Klick-Event, das Text verändert.

HTML + CSS + JavaScript — zusammen

Wir bauen eine kleine Seite mit Überschrift, Absatz und Button. CSS kümmert sich um Abstände und Farben, JavaScript reagiert auf den Klick.

  • HTML: Grundgerüst mit IDs für Button und Absatz.
  • CSS: Schrift, Abstände und Button-Style.
  • JS: addEventListener("click", ...) ändert den Absatztext.
