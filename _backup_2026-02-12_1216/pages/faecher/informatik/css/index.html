CSS

Grundideen: Selektoren, Farben, Hintergründe und Abstände.

Wichtige Bausteine

Selektoren wählen Elemente aus. Mit color und background steuerst du Farben. margin schafft Außenabstand, padding Innenabstand.

Mini-Beispiel: Hintergrundfarbe + Bild

body {
  background-color: #f0f4ff;
  background-image: url('beispielbild.jpg'); /* Nur als Beispiel */
  color: #1b2a3a;
}

h1 {
  margin-bottom: 12px;
  padding: 8px;
}

Das Bild ist nur angedeutet – du kannst später ein echtes Asset einfügen.
