Lektion 02: HTML & JavaScript – Erste Interaktivität

HTML-Struktur mit ersten DOM-Zugriffen verbinden und Klicks reagieren lassen.

Lernziele

  • HTML-Grundgerüst sicher erkennen und aufbauen
  • DOM-Elemente mit querySelector auswählen
  • Ereignisse mit addEventListener verarbeiten
  • Textinhalt dynamisch mit textContent ändern
  • Kleinen Playground nutzen, um HTML & JS direkt auszuprobieren

Wiederholung: HTML-Grundgerüst

Jede Seite startet mit <!DOCTYPE html> und einem <html lang="de">-Element. Im <head> stehen Meta-Angaben, Titel und Styles, im <body> alle sichtbaren Inhalte wie Überschriften, Absätze und Buttons.

Wichtige Tags: <h1> bis <h6> für Überschriften, <p> für Text, <button> für Klickflächen und <div> für Container.

JavaScript Basics

  • DOM: Der Document Object Model bildet die HTML-Struktur als Baum ab, auf den JS zugreift.
  • querySelector: Wählt ein Element per CSS-Selektor, z. B. document.querySelector("#titel").
  • addEventListener("click", ...): Reagiert auf Ereignisse. Beispiel: button.addEventListener("click", handleClick).
  • textContent / innerText: Aktualisiert sichtbaren Text, z. B. ausgabe.textContent = "Hallo!".

Mini-Beispiel

<button id="changeText">Klicke mich</button>
<p id="ziel">Noch kein Klick.</p>

<script>
  const button = document.querySelector("#changeText");
  const ziel = document.querySelector("#ziel");
  button.addEventListener("click", () => {
    ziel.textContent = "Du hast geklickt!";
  });
</script>

Übungen
