Hello World: Überschrift, Button, Absatz – und ein Klick ändert den Text.

Lernziele

  • HTML-Grundgerüst mit Überschrift, Button und Absatz.
  • Einfache CSS-Stile anwenden.
  • Mit JavaScript per Klick den Text verändern.

HTML

Wir brauchen eine Überschrift, einen Absatz und einen Button mit IDs, damit JS sie ansprechen kann.

CSS

Sanfte Hintergründe, ein klarer Button-Stil und etwas Abstand lassen das Mini-UI ordentlicher wirken.

JavaScript

querySelector findet den Button, addEventListener reagiert auf Klick, textContent setzt den neuen Text.

Playground

Bearbeite HTML/CSS/JS, klicke auf „Run“ und sieh dir das Ergebnis in der Vorschau an.
