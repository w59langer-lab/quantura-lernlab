Projekt 01 — Uhr

Digitale Uhr (HTML/CSS/JS)

Demo

Einfache digitale Uhr. Läuft automatisch.

Aufbau

HTML

Struktur mit Containern für Zeit und Datum.

CSS

Layout, Hintergrund, Typo und kleine visuelle Effekte.

JS

Berechnet Zeit/Datum, aktualisiert jede Sekunde.

Erklärung

1) HTML enthält zwei Elemente: Zeit und Datum.
2) CSS sorgt für Lesbarkeit und einfache Karte.
3) JS liest aktuelle Zeit, formatiert sie und schreibt sie in das DOM.

JS

// RU: Обновляем часы каждую секунду
// DE: Uhr jede Sekunde aktualisieren
// EN: Update clock every second
function tick() {
  const now = new Date();
  const time = now.toLocaleTimeString("de-DE", {
    hour: "2-digit",
    minute: "2-digit",
    second: "2-digit",
  });
  const date = now.toLocaleDateString("de-DE", {
    weekday: "long",
    year: "numeric",
    month: "long",
    day: "numeric",
  });

  document.getElementById("time").textContent = time;
  document.getElementById("date").textContent = date;
}

// RU: Старт сразу и повторяем каждую секунду
// DE: Direkt starten und jede Sekunde wiederholen
// EN: Start immediately and repeat every second
tick();
setInterval(tick, 1000);
